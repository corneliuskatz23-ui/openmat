# OPENMAT 

Openmat – Projektbeschreibung

Openmat ist eine Webanwendung für Jiu-Jitsu- und Grappling-Praktizierende, die es ermöglicht, Gyms in der Nähe zu finden und zu besuchen. Nutzer können Informationen über verschiedene Trainingsorte einsehen, z. B. Adresse, Trainingszeiten und angebotene Kurse.

Die Anwendung verwaltet folgende Daten:

Gym-Informationen (Name, Standort, Trainingszeiten, Kontaktdaten)
Optional: Nutzerbewertungen oder Kommentare

Ziel: Den Zugang zu lokalen Trainingsmöglichkeiten erleichtern und eine zentrale Plattform für Kampfsportler schaffen.

Seitenstruktur (Beispiel):

Startseite: Übersicht aller Gyms
Detailseite: Einzelne Gym-Infos
Kontakt/Info: Ansprechpartner oder Impressum
